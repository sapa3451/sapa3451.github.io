Save the USPS!

This project entailed going through thousands of data points relating to the USPS and finding the most efficient way to sort this data. Skills that were used in this program include linked lists, binary search trees, and hash tables (chaining, linear probing, and quadratic probing).

Battleship Trivia

This project is currently in the works. This game is aimed to be a learning experience for all players. You play a game of battleship, but in order to defend against attacks or have successful attacks, you must correctly answer trivia questions.

Boulder Valley School District Consulting Project

In this project, I worked with Boulder Valley School District to analyze the No Student Hungry Weekend Bag Program. Within my report, I suggested efficiency improvements and funding opportunities as well as presented the findings to board members of BVSD.
